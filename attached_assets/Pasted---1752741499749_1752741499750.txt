ПО для дрон-постаматов
IoT

Hardware

О проекте: Пилотный проект завода гражданских дронов «Транспорт будущего» и сервиса доставки «Самокат». Суть проекта — автоматический постомат, который умеет не только автономно выдавать посылки, но и принимать посылки с дрона. Первый в России проект подобного рода. Протестировано в работе первым лицом государства.

Нами сделано: ПО для встроенного компьютера постомата, объединяющее фронтенд (на встроенном планшете постомата), внешние сервисы (API самоката и API «дронопортов») и PLC-контроллеры в единую систему. ПО управляет всей логикой и всеми функциями постомата, включая выдачу посылок и приём посылок с дронов. Также создан прототип интерфейса постомата для встроенного планшета.

О проекте: Personal Relationships Management System — система для управления персональными связями и эффективного нетворкинга. Позволяет объединить все контакты клиента (контакты в телефоне, друзей в социальных сетях, подписчики телеграм каналов клиента и т.п.) в единую базу "карточек", автоматически обогатить карточки людей информацией из открытых источников (OSINT), а также с помощью ИИ осуществлять интеллектуальный поиск людей по произвольному запросу.
